## Quote Generator
Get quotes from the API and picks a quote randomly by clicking on the new quote button.

## Description
It covers these concepts:
- Querying, Manipulating the DOM.
- Handling DOM events.
- Fetch API.
- Twitter API integration.
- Font Awesome.
- Google Fonts.
- Favicons.
- Animations.
- Media Queries.
- Pseudo Classes.
